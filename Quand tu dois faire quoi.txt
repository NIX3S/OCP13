Quand tu dois faire quoi
Si tu changes uniquement le code Angular (TS/HTML/CSS)

Tu fais :

bash
cd chess-agent-frontend
ng build
Puis, depuis la racine du projet (où est ton docker-compose.yml) :

bash
docker compose build frontend
docker compose up -d frontend
→ Pas de npm install nécessaire si les dépendances n’ont pas changé.

Si tu changes package.json / ajoutes un package

Là oui :

bash
cd chess-agent-frontend
npm install
ng build
docker compose build frontend
docker compose up -d frontend
Si tu changes seulement le backend

Tu rebuild seulement le backend :

bash
docker compose build backend
docker compose up -d backend
→ Le frontend reste intact, pas besoin de le rebuild Angular.
